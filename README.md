# 📅 Interactive Calendar Component

A polished, responsive **wall-style calendar component** built with **React / Next.js + Tailwind CSS**, designed to replicate a physical calendar experience while delivering modern interactivity.

---

## 🚀 Overview

This project translates a **static wall calendar design** into a fully interactive UI component.

It focuses on:

* Clean visual hierarchy
* Smooth user interactions
* Thoughtful UX decisions
* Responsive design across devices

---

## ✨ Features

### 🖼 Wall Calendar Aesthetic

* Hero image header acting as visual anchor
* Clean layout mimicking a physical hanging calendar
* Subtle shadows, spacing, and typography for realism

---

### 📆 Interactive Date Selection

* Click any date to select it
* Clearly highlighted active date
* Smooth transition animations between months

---

### 🔄 Animated Navigation

* Month transitions include **side-slide animation**
* Year changes include a subtle **flip effect**
* Selected dates respond with **scale + highlight feedback**

---

### 📝 Notes Integration

* Notes panel linked to selected date
* Notes are persisted using `localStorage`
* Visual indicators show which dates contain notes

---

### 📊 UX Enhancements

* "Today" vs "Selected Date" clearly displayed
* Weekend highlighting
* Visual feedback on save (pulse animation)

---

### 📱 Fully Responsive

* Desktop: split layout (calendar + notes)
* Mobile: stacked layout with touch-friendly interactions

---

## 🛠 Tech Stack

* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **Icons:** lucide-react
* **State Management:** React Hooks
* **Persistence:** localStorage

## ⚙️ How to Run Locally

```bash
# Clone the repo

cd <intern-project>

npm install

# Run development server
npm run dev
```

## 🧠 Design Decisions

* Prioritized **clarity over complexity**
* Avoided heavy UI libraries to maintain control
* Built custom date logic instead of using external calendar packages
* Focused on **UX feedback loops** (selection, transitions, notes)
